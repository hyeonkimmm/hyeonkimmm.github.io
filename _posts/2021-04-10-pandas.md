## 판다스를 배우는 이유
- big data의 시대 - 방대한 데이터를 저장, 분석하는 능력의 결합이 **data science** 라는 새로운 영역의 출현을 가능하게 함.
- 데이터 분석 업무의 **80~90%** 는 데이터 수집하고 정리하는 일
- 나머지 **10~20%** 가 알고리즘 선택, 모델링 결과를 분석하여 유용한 정보를 뽑아내는 프로세스의 몫이다.
- 판다스 라이브러리는 데이터를 수집하고 정리하는데 최적화된 도구.
    - 오픈소스 (무료)
    - python 기반
    - 데이터 과학의 80~90% 업무를 처리 가능
    - 기본적이면서 아주 중요한 도구

## 판다스 자료구조
1. 시리즈(Series) -> 1차원 배열
2. 데이터프레임(DataFrame) - 2차원 배열

### Series
> index와 value 형태로 존재.
python의  **dictionary**와 비슷한 구조.

- 시리즈 변환
```python
# 딕셔너리 -> 시리즈 변환
pandas.Series(dict)
```

```python
# pandas 불러오기
import pandas as pd

# key:value 쌍으로 딕셔너리를 만들고, 변수 dict_data에 저장
dict_data = {'a' : 1, 'b': 2, 'c': 3}

# 판다스 Series() 함수로 dictionary를 Series로 변환. 변수 sr에 저장
sr = pd.Series(dict_data)

# sr 의 자료형 출력
print(type(sr))
print('\n')
# 변수 sr에 저장되어 있는 시리즈 객체를 출력
print(sr)
```
**Output**
```
<class 'pandas.core.series.Series'>


a    1
b    2
c    3
dtype: int64
```

- 인덱스 구조
```
인덱스 배열: Series객체.index
데이터 값 배열: Series객체.values 
```
이는 python dictionary와 비슷한 것 같다.

#### 시리즈 인덱스
```python
# -*- coding: utf-8 -*-
import pandas as pd

#리스트를 시리즈로 변환하여 변수 sr에 저장
list_data = ['2019-01-02', 3.14, 'ABC', 100, True]
sr = pd.Series(list_data)
print(sr)
```
**Output**
```
0    2019-01-02
1          3.14
2           ABC
3           100
4          True
dtype: object
```

```python
idx = sr.index
val = sr.values
print(idx)
print('\n')
print(val)
```
**Output**
```
RangeIndex(start=0, stop=5, step=1)


['2019-01-02' 3.14 'ABC' 100 True]
```

index와 values 속성 활용해서 시리즈데이터의 배열 값을 불러 올 수 있다.

#### 시리즈 원소 선택
```python
#튜플을 시리즈로 변환(인덱스 옵션 지정)
tup_data = ('영인','2021-04-21','여',False)
sr = pd.Series(tup_data, index=['이름','생년월일','성별','학생여부'])
print(sr)
```

**Output**
```
이름              영인
생년월일    2021-04-21
성별               여
학생여부         False
dtype: object
```
---
```python
# 원소 선택
print(sr[0]) # sr의 1번째 원소 선택(정수형 위치 인덱스)
print(sr['이름']) # '이름' 라벨을 가진 원소 선택
```

**Output**
```
영인
영인
```
---
```python
# 여러 개의 원소를 선택(인덱스 리스트 활용)
print(sr[[1,2]])
print(sr[['생년월일','성별']])
```
**Output**
```
생년월일    2021-04-21
성별               여
dtype: object
생년월일    2021-04-21
성별               여
dtype: object
```
