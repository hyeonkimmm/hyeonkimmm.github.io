## The Check ETL Application
- [Github 주소](https://github.com/mattcho/tc-etl-v0)
- Python flask 기반 application
- Elastic Beanstalk 기반으로 개발
- [Elastic Beanstalk 참고](https://docs.aws.amazon.com/ko_kr/elastic-beanstalk/index.html)

---
### Purpose 
- The Check App에서 operational database를 Extract함
- csv file로 변환
- AWS S3 buckets에 csv files를 load
- 매일 한국시간 기준 5:00 am에 실행되어야 함
- 이후 email 발송

---
### Prerequisites
- install `pipenv`
- install the EB CLI
