## AWS Elastic Beanstalk
- AWS 클라우드에서 애플리케이션을 신속하게 배포하고 관리하는 **Paas**(Platform as a Service)
- Application 업로드 시 EB에서 용량 프로비저닝, 로드 밸런싱, 조정, app status monitoring 자동 처리
- Elastic Beanstalk 콘솔 , AWS CLI, EB전용 CLI인 eb를 활용해 상호 작용 가능
